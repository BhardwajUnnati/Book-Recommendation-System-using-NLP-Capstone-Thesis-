# Book-Recommendation-System-using-NLP-Capstone-Thesis-
## Project Overview
This project presents a hybrid book recommendation system combining Collaborative Filtering, Content-Based Filtering, and NLP techniques using transformer-based embeddings (RoBERTa, BERT, XLNet).
The system is designed to improve recommendation relevance, personalization, and semantic understanding of book content and user preferences.

# Key Features
- Hybrid recommendation system (CF + CBF + NLP)
- Transformer-based embeddings for semantic analysis
- Handles cold-start and overspecialization issues
- Uses both structured (metadata) and unstructured (reviews) data
# Dataset
- Goodreads dataset
~93k books + ~2.3M reviews

# Used for:

- CF → user interactions
- CBF → book metadata + descriptions
